# EC2 Cost Optimization Lab (M2.10)

This repository contains the documentation and tools for analyzing and optimizing AWS EC2 costs. The project focuses on cost visibility, identifying waste, and implementing lifecycle best practices.

## File Structure

- `COST-ANALYSIS.md`: Detailed audit of current AWS configurations and theoretical monthly costs.
- `OPTIMIZATION-PLAN.md`: Strategic plan for reducing expenditures through right-sizing and scheduling.
- `IMPLEMENTATION-LOG.md`: Chronological log of optimization actions taken (CLI checks, volume migrations).
- `SUMMARY-REPORT.md`: Final executive summary of achieved and projected savings.

- **cost-calculator.csv**: Comparative spreadsheet showing "Current" vs "Optimized" cost scenarios.
